# Usable Security Blog

Static site, ready to deploy. Just upload everything as-is.

Contents:
- index.html ............ homepage
- *.html ................ 8 article pages
- images/ ............... all article screenshots (linked with relative paths)
- favicon.svg ........... site icon (linked on every page)

Homepage company logos are inline SVG, so they need no files.
Author avatars are shown as lettered circles (no image files required).

## Deploy to GitHub Pages
1. Commit and push the whole folder (including images/ and favicon.svg).
2. Repo Settings -> Pages -> deploy from your branch, root folder.
Everything uses relative paths, so it works from any path.
